# hoko
Python library for querying Bitcoin exchange tickers, order books. 
Forked from https://github.com/dursk/bitcoin-price-api.

##Exchanges included##
 - BTC-E
 - CoinBase
 - CoinSetter
 - Bitfinex
 - Bitstamp
 - Huobi
 - OKCoin
 - Quoine
 - MexBT

##Data Providers##
 - CoinDesk
 - Coinapult
